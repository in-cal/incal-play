# InCal Play [![version](https://img.shields.io/badge/version-0.1.9-green.svg)](https://in-cal.org)

Play Framework extended with basic readonly/crud controllers, deadbolt-backed security, json formatters, etc. 

#### Installation

All you need is **Scala 2.11**. To pull the library you have to add the following dependency to *build.sbt*

```
"org.in-cal" %% "incal-play % "0.1.9"
```

or to *pom.xml* (if you use maven)

```
<dependency>
    <groupId>org.in-cal</groupId>
    <artifactId>incal-play_2.11</artifactId>
    <version>0.1.9</version>
</dependency>
```
